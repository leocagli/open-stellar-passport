## [1.0.0] - 2025-02-14

### Added
- Initial release of Open Stellar Passport.
- Groth16 ZK circuits for agent identity and proof-of-funds.
- Soroban contracts for proof verification and attestation registry.
- TypeScript SDK for client-side proving and contract interaction.
- Demo frontend built with Vite and React 19.
